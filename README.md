Welcome to the Personal Budget Tracker!
    This program helps you manage your finances by tracking income, expenses, and monthly budgets.

Requirements:
    Python 3.6+

How to Run:
    python budget_tracker.py

First Run:
    If you see "No saved data found. Starting fresh." â€” that's normal. A save file (budget_data.txt) will be created automatically when you exit.

Features Overview:
    1. Add Income: Record money coming in (e.g., salary, gifts).
    2. Add Expense: Record your spending into 5 main categories:
        - Food, Transportation, Entertainment, Bills, Other.
    The program will warn you if you are close to or over your budget.
    3. View All Transactions: See a chronological list of all your recorded income and expenses.
    4. Set Budget: Define a monthly spending limit for any expense category.
    5. View Budget Summary: View your total income, total expenses, current balance, and spending vs budget breakdown.
    6. Generate Report: (Coming Soon)
    7. Save & Exit: Close the application.

Getting Started:
    - It is recommended to Set Budgets (Option 4) first.
    - Navigate the menu by typing the number corresponding to your choice and pressing Enter.
    - Follow the prompts to add your financial transactions.

Notes:
    Always exit with Option 7 to save your data.
    To reset all data, delete budget_data.txt and restart.
