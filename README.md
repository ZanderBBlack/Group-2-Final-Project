Personal Budget Tracker

Requirements:
Python 3.6+

How to Run:
python budget_tracker.py

First Run:
If you see "No saved data found. Starting fresh." — that's normal. A save file (budget_data.txt) will be created automatically when you exit.

Menu Overview:
    - Add Income
    - Add Expense — categorized under Food, Transportation, Entertainment, Bills, or Other
    - View All Transactions
    - Set Budget — set a monthly spending limit per category
    - View Budget Summary
    - Generate Report — includes net income and savings rate
    - Save & Exit

Notes:
Always exit with Option 7 to save your data.
Set a budget (Option 4) to get spending warnings as you add expenses.
To reset all data, delete budget_data.txt and restart.
